# CUI_JAVA_PROJECTS
Console based Java project using Collections + MySQL + JDBC with CRUD operations.




This repository contains a Console Based Application developed in Java using Collections, MySQL, and JDBC.
The project demonstrates complete end-to-end CRUD operations with database persistence, where the core business logic is managed using Java Collections while the permanent data storage is handled through MySQL database.

Key Highlights:

Java Core + Collections Framework implementation

Menu driven console interaction

MySQL based backend storage

JDBC used for database connectivity and operations

Clean modular code structure suitable for learning + reference usage

This repo is ideal for Java learners, mini project submissions, and backend fundamentals practice.
